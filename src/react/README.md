# React migration track

Этот каталог готовит перенос Live Scanner на React + TypeScript без замены текущей опубликованной static-версии.

## Что уже перенесено

- Отдельная точка входа `react/index.html`.
- Корневой компонент `App`.
- Компонентная оболочка `AppShell`.
- Первый React-экран `LiveScannerView`.
- Экран `SignalListView` для найденных сигналов.
- Экран `PatternLabView` для паттернов, условий и базовой аналитики.
- Экран `HistoryView` для истории событий и фильтров результата.
- Переключение между React-разделами через desktop sidebar и mobile bottom navigation.
- Хук `useFootballLabData`, который берет существующие mock-данные и прогоняет их через TypeScript Pattern Engine.
- Переиспользуются текущие стили из `styles.css`, чтобы дизайн не расходился между версиями.

## Как запускать

После установки зависимостей:

```powershell
npm run dev:react
```

Открыть:

```text
http://127.0.0.1:5173/react/
```

## Правило миграции

Текущий сайт на `index.html` остается рабочей production-версией, пока React-версия не повторит основные экраны:

1. Сканер матчей.
2. Найденные сигналы.
3. Паттерны.
4. История.
5. Аналитика.
6. Профиль команды.
7. Настройки и идеи.

После этого можно будет переключать GitHub Pages на собранную React-версию.
