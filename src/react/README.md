# React migration track

Этот каталог содержит основную React + TypeScript-версию Live Scanner.

## Что уже перенесено

- Корневая точка входа `index.html`.
- Корневой компонент `App`.
- Компонентная оболочка `AppShell`.
- Первый React-экран `LiveScannerView`.
- Экран `SignalListView` для найденных сигналов.
- Экран `PatternLabView` для паттернов, условий и базовой аналитики.
- Экран `HistoryView` для истории событий и фильтров результата.
- Экран `AnalyticsView` для эффективности паттернов и качества данных.
- Профиль команды при клике на команду в React-сканере.
- Экран `SettingsView` для режима данных, защищённых Telegram-уведомлений и API-адаптеров.
- Экран `ProfileView` для личного профиля, публичной заготовки и социального доверия.
- Экран `IdeasView` для идей, feedback-очереди, статусов и приоритетов.
- Переключение между React-разделами через desktop sidebar и mobile bottom navigation.
- Хук `useFootballLabData`, который берет существующие mock-данные и прогоняет их через TypeScript Pattern Engine.
- Переиспользуются текущие стили из `styles.css`, чтобы дизайн не расходился между версиями.

## Как запускать

После установки зависимостей:

```powershell
npm run dev:react
```

Открыть:

```text
http://127.0.0.1:5173/react/
```

## Правило миграции

React-версия повторяет основные экраны MVP:

1. Сканер матчей.
2. Найденные сигналы.
3. Паттерны.
4. История.
5. Аналитика.
6. Профиль команды.
7. Настройки, профиль и идеи.

GitHub Pages публикует собранную React-версию из `dist-react`.
